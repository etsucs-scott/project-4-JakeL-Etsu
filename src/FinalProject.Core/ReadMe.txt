This is a web based team builder based off of one of my favorite games Honkai Star Rail. It is built with blazor and C#.
You can browse 1.0 characters, build teams of 4, and save your teams.

You can run it by having .NET 10.0 installed
Then run the project with : dotnet run --project src/FinalProject.UI

Then you have to go to your browser if it doesnt automaticall open and go to: http://localhost:5090

You can run the tests with dotnet test, this should run all 12 tests which should hopefully pass for you since they did for me

For the Tech Stack, I used C#, Blazor Server, and JSON 

Key features: Browse list of characters by element, path, or rarity.
Create custom teams with custom team names
Remove or add characters from a team

All of the data is stored in a JSON file located under the data folder.
